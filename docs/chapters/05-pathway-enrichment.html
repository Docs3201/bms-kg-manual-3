6  Pathway & Functional Enrichment

KEGG, GO, GSEA, and ORA results stored in the KG

6.1 Overview

Pathway enrichment results are pre-computed and stored directly in the KG as edges between pathway nodes and cohort, network, or factor nodes. You do not need to run enrichment yourself—you query the results.

6.2 Enrichment Methods

Method Full name Input What it finds
GSEA Gene Set Enrichment Analysis Ranked gene list (by log2FC or MOFA weight) Pathways with coordinated shifts across the full ranked list
ORA Over-Representation Analysis Binary gene list (DEGs or module members) Pathways with more hits than expected by chance

Both methods are applied to:

  • DEG cohorts (per molecule, per condition)
  • WGCNA modules (significant modules only)
  • MOFA factors (top 80th percentile features per factor per omics layer)

Databases used: KEGG Pathways, Gene Ontology Biological Process (GO:BP), Gene Ontology Molecular Function (GO:MF), Gene Ontology Cellular Component (GO:CC).

6.4 Key Biological Themes

Results from DEG, WGCNA, and MOFA all converge on consistent pathway themes:

Pro-Productivity Pathways

Pathway Source NES / Significance

Anti-Productivity Pathways

Pathway Source NES / Significance
Cell cycle MOFA F1 GSEA −3.2 ★★★
DNA replication MOFA F1 GSEA −3.0 ★★★
Homologous recombination MOFA F1 GSEA −2.9 ★★★
Fanconi anemia pathway MOFA F1 GSEA −2.7 ★★★
Mismatch repair MOFA F1 GSEA −2.6 ★★★
Tip

Biological interpretation: High producers allocate resources to ER protein folding and secretion (UPR / secretory pathway upregulated) while suppressing cell proliferation programs (cell cycle / DNA replication downregulated). This is consistent with the canonical productivity vs. growth trade-off in CHO cells.
